# Circuitos digitales en Multisim

Circuitos diseñados en **Multisim**, organizados por tipo de circuito.

> Los ficheros `.ms14` requieren Multisim para abrirse; son binarios y no se
> visualizan en GitHub.

## combinacionales
Sumadores, sumador/restador, multiplexores, transcodificadores, conversor
binario→BCD, sistema de votación y decodificadores (incluye las versiones de
práctica y de examen de cada uno).

## secuenciales
Contadores y registros (con sus versiones de práctica y examen).
